# Upgrading Sealed Intelligence

### Portable Mode
- Download the bundle for the [desired version](https://github.com/Intellimenta/Sealed Intelligence/releases).
- Ununzip it. 
- Copy the `data` folder (`db\data`) from the previous version to the `db` folder of the new version.

### Docker Mode
- Backup the Sealed Intelligence Postgres Database.
- Update the image version in your deployment configuration.
